════════════════════════════
SCOPE CLASSIFICATION (required field: "scope")
════════════════════════════

You MUST classify the student's question into one of three scopes:

"global" → student lacks the whole concept
  signals: "don't understand", "explain everything", "what is", "samajh nahi aaya",
  "explain from scratch", "never studied this", "concept clear nahi hai",
  "explain this" with full-concept image, intent = "full_explanation"

"local" → student has a specific wrong belief about one part of the concept
  signals: "why does X happen", "I thought X was Y", "isn't it supposed to be",
  "why not", "how can", "shouldn't it be", specific contradiction stated,
  intent = "specific_confusion"

"micro" → student is asking about one symbol, one step, one constant, one line, or one small part
  signals: "why cosθ", "why this sign", "where does this constant come from",
  "what does this symbol mean", "why 2 here", "I marked this part",
  marked region present with a short question, derivation questions,
  formula origin questions

Default: "local" if uncertain.

════════════════════════════
SIMULATION_NEEDED CLASSIFICATION (required field: "simulation_needed")
════════════════════════════

Set simulation_needed = false when ANY of these are true:
  - student asks for a derivation or proof
  - student asks where a formula comes from mathematically
  - student says "I got wrong answer" or "check my work" or "where did I go wrong"
  - student asks for a definition only
  - student asks to compare two concepts textually
  - student asks about history or who discovered
  - question is purely algebraic manipulation

Set simulation_needed = true when ANY of these are true:
  - confusion is about something that moves or changes
  - confusion is about direction, spatial relationship, or what something looks like
  - student has a wrong belief about physical behavior
  - student marked a diagram asking about what is happening physically
  - scope is "global" (full concept explanation always needs simulation)

Default: true if uncertain.
