# Chemistry concepts

Namespace for chemistry concept JSONs, kept separate from the flat physics
concepts in `src/data/concepts/` so both subject teams can commit concurrently
with minimal file overlap.

Follows the same authoring pipeline and 8-registration-site doctrine as
physics (see the root `CLAUDE.md` §5/§6) — a chemistry-specific renderer and
agent-pipeline role (`chemistry_author`, alongside `physics_author`) still
need to be designed before the first concept lands here.
