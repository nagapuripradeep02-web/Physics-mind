## Cross-Pattern Observations

### 5 Modal Failure Patterns

1. **Sign-convention errors** (~30% of all cognitive_error_targets): students mis-apply +/− conventions in kinematics, circuit-direction, refraction-sign-rules, Lenz's law. **Countermeasure:** mandatory sign-explainer nano per atomic that uses signs.

2. **Log-scale / non-linear intuition failures** (~15%): students apply linear intuition to dB, magnitudes, half-life (exponential), photon-energy-vs-frequency. **Countermeasure:** explicit log-scale nano with worked +3dB ≈ 2× example.

3. **Frame-of-reference asymmetry** (~12%): students assume Galilean-invariance where it fails (sound Doppler vs EM Doppler; centrifugal in rotating frame; relative motion across media). **Countermeasure:** explicit "which frame" nano per such atomic.

4. **Microscopic-vs-macroscopic confusion** (~15%): students conflate per-atom and bulk-property (ferromagnetism, kinetic theory, semiconductor doping, photoelectric per-photon). **Countermeasure:** explicit "atom-level vs material-level" visual.

5. **Scale-invariance failures** (~10%): students assume small-amplitude / linear regime extends (Hooke's law, superposition, ideal-gas, thin-lens). **Countermeasure:** explicit "when this assumption breaks" callout nano.
